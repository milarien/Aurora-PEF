# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this repository,  
please **do not open a public issue**.

Instead, report it by email to:

**margaret.stokes.ai@gmail.com**

Please include:
- a clear description of the issue
- the file or component involved
- steps to reproduce (if applicable)

## Scope

This repository contains research materials and demonstration code.  
It is not a production system and is provided as-is.

## Response

Security reports will be reviewed as time permits.  
No response time or fix timeline is guaranteed.
